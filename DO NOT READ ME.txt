=== INSTALLATION GUIDE FOR NEVER MULTITOOL ===
EDUCATIONAL PURPOSES ONLY!

This tool requires Python 3, Nmap, hping3, macchanger, alsa-utils, dnsutils, 
colorama and the pyfiglet module.
Install them according to your distribution:

--- Debian / Ubuntu / Kali Linux ---
sudo apt update
sudo apt install python3 nmap hping3 macchanger alsa-utils dnsutils
pip install pyfiglet colorama

--- Fedora ---
sudo dnf install python3 nmap hping3 macchanger alsa-utils bind-utils
pip install pyfiglet colorama

--- Arch Linux ---
sudo pacman -S python nmap hping macchanger alsa-utils bind-tools
pip install pyfiglet colorama

--- openSUSE ---
sudo zypper install python3 nmap hping macchanger alsa-utils bind-utils
pip install pyfiglet colorama

--- Void Linux ---
sudo xbps-install -S python3 nmap hping macchanger alsa-utils bind-utils
pip install pyfiglet colorama

--- Gentoo ---
sudo emerge --ask dev-lang/python net-analyzer/nmap net-analyzer/hping 
net-analyzer/macchanger media-libs/alsa-lib net-dns/bind-tools
pip install pyfiglet colorama

After installation, run the tool with:
python3 NEVER.py
