=== INSTALLATION GUIDE FOR NEVER MULTITOOL ===
EDUCATIONAL PURPOSES ONLY!

This tool requires Python 3, Nmap and the pyfiglet module.
Install them according to your distribution:

--- Debian / Ubuntu / Kali Linux ---
sudo apt update
sudo apt install python3 nmap
pip install pyfiglet

--- Fedora ---
sudo dnf install python3 nmap
pip install pyfiglet

--- Arch Linux ---
sudo pacman -S python nmap
pip install pyfiglet

--- openSUSE ---
sudo zypper install python3 nmap
pip install pyfiglet

--- Void Linux ---
sudo xbps-install -S python3 nmap
pip install pyfiglet

--- Gentoo ---
sudo emerge --ask dev-lang/python net-analyzer/nmap
pip install pyfiglet

After installation, run the tool with:
python3 never_multitool.py
