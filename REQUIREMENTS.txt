=== INSTALLATION GUIDE FOR NEVER MULTITOOL ===
EDUCATIONAL PURPOSES ONLY!

This tool requires Python 3, Nmap, hping3, macchanger, alsa-utils, dnsutils, 
netcat-openbsd, curl, sublist3r, colorama and the pyfiglet module.
Install them according to your distribution:

--- Debian / Ubuntu / Kali Linux ---
sudo apt update
sudo apt install python3 nmap hping3 macchanger alsa-utils dnsutils netcat-openbsd curl
pip install pyfiglet colorama sublist3r

--- Fedora ---
sudo dnf install python3 nmap hping3 macchanger alsa-utils bind-utils nmap-ncat curl
pip install pyfiglet colorama sublist3r

--- Arch Linux ---
sudo pacman -S python nmap hping macchanger alsa-utils bind-tools openbsd-netcat curl
pip install pyfiglet colorama sublist3r

--- openSUSE ---
sudo zypper install python3 nmap hping macchanger alsa-utils bind-utils netcat-openbsd curl
pip install pyfiglet colorama sublist3r

--- Void Linux ---
sudo xbps-install -S python3 nmap hping macchanger alsa-utils bind-utils netcat curl
pip install pyfiglet colorama sublist3r

--- Gentoo ---
sudo emerge --ask dev-lang/python net-analyzer/nmap net-analyzer/hping net-analyzer/macchanger media-libs/alsa-lib net-dns/bind-tools net-analyzer/netcat net-misc/curl
pip install pyfiglet colorama sublist3r

After installation, run the tool with:
python3 NEVER.py
